# Playwright Automation Project

This repository contains Playwright automation scripts and examples.

## Project Structure

- `01_Basic.js`: Basic Playwright script.
- `02_Verify_Setup.js`: Script to verify the Playwright setup.
- `03_Step_by_step.js`: Step-by-step automation example.
- `04_hot_code.js`: Additional automation scenarios.

## Prerequisites

- Node.js installed
- Playwright installed

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/rajuljain-hub/Playwright.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

## Running Tests

To run the scripts, use Node.js:

```bash
node 01_Basic.js
```
